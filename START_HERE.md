## 🚀 3-Step Deployment (30 minutes)

### Step 1: Upload to GitHub (10 min)

**Easiest: Use Emergent's "Save to GitHub"**
1. Click "Save to GitHub" in Emergent
2. Create repo: `digital-globexim-dashboard` (Public)
3. Select all files from `/app/github_upload/`
4. Push!

**Alternative: Manual Upload**
1. Create GitHub repo: https://github.com/new
2. Upload files from `/app/github_upload/`
3. Done!

---

### Step 2: Setup MongoDB Atlas (15 min)

1. Go to https://cloud.mongodb.com
2. Sign up (FREE)
3. Create cluster (M0 Free tier)
4. Create user & password
5. Set network access: 0.0.0.0/0
6. Get connection string
7. **SAVE PASSWORD!**

**Detailed guide**: Open `DEPLOYMENT_CHECKLIST.md`

---

### Step 3: Deploy to Streamlit Cloud (5 min)

1. Go to https://share.streamlit.io
2. Sign up with GitHub
3. New app → Select your repo
4. Main file: `Home.py`
5. Add secrets (MongoDB connection)
6. Deploy!

**Your app will be live**: `https://your-app.streamlit.app`
