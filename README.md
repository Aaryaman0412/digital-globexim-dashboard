# Digital GLOBEXIM - Medical Device Trade Dashboard

India's comprehensive medical device import-export analytics platform.

## 🎯 Features

- **Trade Overview** - Import/Export trends and KPIs
- **Top Traded Devices** - Rankings and analysis
- **Import Risk Mapping** - HHI concentration index
- **Import Reliance** - Country-wise dependency
- **Export Performance** - Export tracking

## 📊 Dataset

- **165,337 records** of medical device trade data
- **155 products** across medical device categories
- **223 countries** of import/export activity
- **2023-2024** time period

## 🚀 Quick Start

1. Upload files to GitHub
2. Deploy to Streamlit Cloud
3. Share your dashboard!

## 🛠️ Tech Stack

- **Streamlit** - Web framework
- **Pandas** - Data processing
- **Plotly** - Interactive charts
- **Python 3.11+**
