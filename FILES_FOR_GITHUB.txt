## 🚀 Quick Upload Guide

### Option 1: Emergent's Save to GitHub
1. Use Emergent's built-in GitHub integration
2. Select all files from /app/streamlit_app/
3. Exclude .streamlit/secrets.toml
4. Push to new repository

### Option 2: Manual Upload via GitHub Web
1. Go to your GitHub repository
2. Click "Add file" → "Upload files"
3. Drag and drop all files listed above
4. Commit changes

### Option 3: Git Command Line
```bash
cd /app/streamlit_app
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/digital-globexim-dashboard.git
git push -u origin main
```
